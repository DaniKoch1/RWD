Exercise, Fourth RWD Session

Add external style-sheet to your Star Pizza website.

  1. Make a new folder for this session
  2. Copy your Star Pizza homepage into the new folder
  3. Copy the ccs-folder along with the styles.css-file from studienet into the folder containing the Star Pizza homepage
  4. Add a link in the head section of the index.html for the Star Pizza homepage to the copied styles.css-file
  5. Change the font type, size and/or color of the headings (in the copied file)
  6. Change the background color, possibly replacing it with an image
  7. Add a link in the head section of the menu.html for the Star Pizza homepage to the styles.css-file, so both pages uses the same styling
  8. Add visible colored (pick a color) borders around the images on the menu-page
  9. Lower the font-size of the image captions
  10. Improve the look of your webpage with additional styling.
