Exercise, Fifth RWD Session

Working with the CSS box and styling links.

  1. Make a new folder for this session
  2. Copy your Star Pizza homepage into the new folder and work on this copy
  3. Make separate HTML pages for each of your menu categories (pizzas, Pasta, Beverages etc.). Make links to them on the main page.
  4. Add an About page and write a little story about Star Pizza. Include the “About us” page in your links on the main page.
  5. Style each of your pages with a different background image or color. Pick your image so that it matches the content on the page. Cover the whole page by repeating or stretching the image.
  6. Create a style for your links so that they appear as buttons on the main page.
  7. In a new HTML document (not related to the Star Pizza website), create a style that can insert a textbox with borders in another textbox like shown on the image on the right. HINT: To get the text box to be on the right side, use: float: right;
